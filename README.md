# Dungeon Crown

Dungeon Crown is a 2D dungeon adventure game made in Python using Pygame.

## Features

- Class selection system
- Multiple enemies with diffrent attack patterns
- Undertale-style combat system
- Collision map system
- Large scrolling overworld map
- Animated endings with video and audio
- Boss-style attack mechanics

## Technologies

- Python
- Pygame
- OpenCV
- MoviePy

## Controls

### Overworld
- WASD - Move

### Combat
- WASD - Dodge attacks
- SPACE - Attack

## Goal

Explore the dungeon, survive enemy encounters and recover the lost crown.
